Detailed Description

Class for the MQ2 gas sensor.

This sensor detects gas concentration and relays the concentration via the analog pin. If the concentration is past a certain value (configured via the potentiometer on the back), it pulls the digital pin to HIGH, otherwise, it's low.

Note
For more info: https://lastminuteengineers.com/mq2-gas-senser-arduino-tutorial/
